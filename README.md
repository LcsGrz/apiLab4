# apiLab4
API para la materia laboratorio IV

Funcionamiento: 

1) Ejecutar la bd -> mongod --dbpath ./pullers/db
2) Ejecutar script CR4WLER en pullers -> node CR4WL3R.js (recordar estar dentro de la carpeta Pullers)
3) Ejecutar insertarRoles -> node insertarRoles.js (Esperar unos segundos y cerrar con CTRL + C )
4) Ejecutar servidor -> node server.js
5) Ejecutar app de react -> npm start (Recordar estar dentro de LabUI)
